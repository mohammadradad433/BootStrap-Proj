## Features

- Fully responsive layout using **Bootstrap 5 grid system**  
- Fixed **navigation bar** with smooth scrolling  
- **Interactive UI components**: flip cards, accordions, buttons  
- Modern design: gradients, hover effects, shadows, and rounded cards  
- **Font Awesome icons** integrated for social links and visuals  
- Multi-page navigation with consistent header and footer
